## 6. 積分頁面

**功能說明**：
- 顯示當前積分餘額（大字體）
- 顯示即將到期積分和日期
- 交易記錄列表（支援下拉刷新）
- 每筆交易顯示：類型、積分、描述、日期、門市

**交易類型**：
| 類型 | 圖示 | 說明 |
|------|------|------|
| 賺取 | + | 購物消費獲得積分 |
| 兌換 | - | 兌換優惠券扣除積分 |
| 過期 | ⚠️ | 積分過期失效 |
| 調整 | 🔄 | 系統調整 |

## 技術規格

| 項目 | 規格 |
|------|------|
| 框架 | React Native 0.81 + Expo SDK 54 |
| 架構 | MVP (Model-View-Presenter) |
| 導航 | Expo Router (File-based) |
| 狀態管理 | React Context API |
| 國際化 | i18next |
| 安全儲存 | expo-secure-store (JWT Token) |
